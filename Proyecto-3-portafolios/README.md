## Instrucciones Generales
Este proyecto está dividido en cinco repositorios, por motivos de organización y para garantizar el buen funcionamiento de GitHub Pages. 
En primer lugar, se encuentra el repositorio correspondiente a la página general. En esta página se mostrarán cuatro cuadros con los nombres de los integrantes y los enlaces a los portafolios de cada uno. De igual manera, se incluirán los enlaces directos a los portafolios en caso de que la página principal presente algún fallo.
